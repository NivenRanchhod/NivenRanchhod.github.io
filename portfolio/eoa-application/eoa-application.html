The Tesla project I listed in my portfolio is a project I want to share with the community, including a write up on how I completed the project and why I did it. It is based off of the old Tesla range calculator they had on their Model S listing.
The functionality of the email is a showroom slideshow and then the range calculator will change the range appropriately, depending on the model, taking note of the additional animation on the wheels which speed up/down depending on the speed set on the panel. A feature I built into the tool and was my own inclusion and twist on the original tool.
The design, layout and code is all my work (including slicing up the updated 2021 model imagery, which was an exhaustive task) and I'm so proud of being able to take the front end JS reliant calculator, redesign it to suit the updated website and modern day design techniques and then completely code it myself (fallbacks included).
This is a great example of what drives me as a developer, Trying out new techniques and sharing my experiences with others.
